## 网络线路怎么选？三种BGP别搞混

野草云的香港VPS有三种网络线路，初次购买经常搞不清楚，这里简单说一下：

**优质BGP**：中国电信回程走CMI，联通回程走CU/NTT/CMI均衡，移动走Level3。国内访问速度不错，是大多数轻量套餐的默认选项。

**精品BGP**：电信、联通都走CU，移动走CMI。三网骨干网回程，晚高峰时段波动更小，速度更稳定，价格也比优质BGP略贵一些。

**国际BGP**：不含CMI和CU内地路由，线路择优选路，内地延迟较高。适合东南亚跨境电商独立站、面向海外用户的业务，不适合主要服务内地访客的项目。

简单记：**要服务国内用户选优质或精品BGP，做东南亚/欧美外贸选国际BGP，要省钱且追求稳定就上精品BGP**。
